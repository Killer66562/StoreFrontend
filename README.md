# 購物網站排版測試

## 目前使用套件
+ Vue
+ TypeScript
+ Vue-Router
+ Bootstrap
+ Axios
+ Vue-toast

## 怎麼讓它跑起來（測試環境）
1. 下載nodejs和npm
2. 下載這個專案
3. 進到專案根目錄
4. 終端機輸入 "npm install" 安裝所需套件
5. 終端機輸入 "npm run dev" 啟動測試環境

## 功能測試
+ 後端啟動後即可測試登入等相關功能
+ 已完成功能
    + 註冊
    + 登入
    + 登出
    + 列出所有商品
    + 查看自己的商店
    + 查看商品詳細資料
